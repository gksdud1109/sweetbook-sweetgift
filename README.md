# SweetGift 🎁

**"우리의 소중한 순간들을 한 권의 책으로, 가장 다정한 기념일 선물"**

SweetGift는 커플들이 기념일을 위해 흩어진 사진과 짧은 추억 기록, 그리고 진심을 담은 편지를 단 몇 분 만에 한 권의 세련된 실물 앨범으로 만들 수 있도록 도와주는 서비스입니다.

---

## 1. 서비스 소개

### 누구를 위한 서비스인가요? (타겟 고객)
- 기념일 선물을 준비하고 싶지만, 디자인 감각이나 시간이 부족한 분
- 스마트폰 속에 잠들어 있는 연인과의 소중한 사진들을 의미 있게 보관하고 싶은 분
- 단순한 굿즈보다 정성이 담긴 '물성' 있는 선물을 선호하는 분

### 주요 기능 목록
- **기념일 맞춤 설정**: 100일, 1주년 등 다양한 기념일 타입과 날짜 입력
- **감성적 앨범 편집**: 최대 8장의 추억 사진과 일기, 진심을 담은 편지 작성
- **에디토리얼 미리보기**: 실제 책처럼 구성된 표지, 타임라인, 편지 페이지 실시간 렌더링
- **이미지 업로드**: 로컬 파일을 즉시 업로드하고 프리뷰 확인
- **SweetBook 주문 연동**: 제작된 앨범을 실제 인쇄용 도서(Books API)로 생성하고 주문(Orders API) 접수

## 2. 실행 방법

심사자분들의 편의를 위해 **모노레포 루트**에서 단 한 번의 명령어로 모든 개발 환경을 구축할 수 있도록 설정했습니다.

### 설치 및 환경 설정
```bash
# 1. 의존성 설치
pnpm install

# 2. 환경 변수 설정
# 루트의 .env.example을 참고하여 .env 파일을 생성합니다.
cp .env.example .env

# 3. .env 파일에 SweetBook API Key를 입력해주세요.
# SWEETBOOK_API_KEY=your_sandbox_key_here
```

### 실행
```bash
# 프론트엔드(Next.js)와 백엔드(Fastify)를 동시에 실행합니다.
pnpm dev
```
- **Frontend**: `http://localhost:3000`
- **Backend**: `http://localhost:3001` (API 서버 및 정적 파일 서빙)

## 3. 사용한 API 목록

SweetBook의 `Books` 및 `Orders` API를 핵심으로 사용하며, 원활한 UX를 위해 자체 업로드 API를 구축했습니다.

| API 엔드포인트 | 용도 | 비고 |
| :--- | :--- | :--- |
| `POST /api/v1/album-drafts` | 사용자의 입력을 바탕으로 앨범 초안 생성 및 페이지 레이아웃 계산 | 자체 API |
| `GET /api/v1/album-drafts/:id` | 생성된 초안 정보 조회 및 미리보기 렌더링용 데이터 확보 | 자체 API |
| `POST /api/v1/uploads` | 사용자가 선택한 사진을 서버 로컬 디스크에 업로드 및 URL 반환 | 자체 API |
| `POST /api/v1/books` | 완성된 초안을 **SweetBook Books API**와 연동하여 도서 생성 | **필수 API** |
| `POST /api/v1/orders` | 생성된 도서를 **SweetBook Orders API**와 연동하여 주문 접수 | **필수 API** |

## 5. 설계 의도

### 왜 이 서비스를 선택했나요? (비즈니스 가능성)
- **높은 전환율**: 커플 선물 시장은 '감성'과 '정성'이 핵심입니다. SweetGift는 사용자가 최소한의 노력으로 결과물을 즉시 확인하게 함으로써(Preview-First), 실제 주문(Conversion)까지의 장벽을 크게 낮춥니다.
- **확장성**: 현재는 기념일 앨범에 집중하고 있지만, 여행, 육아, 졸업 앨범 등 테마별 템플릿 확장이 매우 용이한 구조입니다.

### 더 시간이 있었다면 추가했을 기능
- **드래그 앤 드롭 정렬**: 추억 페이지의 순서를 사용자가 자유롭게 바꿀 수 있는 UI
- **AI 문구 추천**: 사진 분석을 통해 해당 장면에 어울리는 다정한 멘트 자동 생성
- **실시간 협업**: 연인이 함께 접속하여 앨범을 공동 편집하는 기능

### 안내 사항
- 본 프로젝트는 **Next.js 14 (App Router)**와 **Fastify**를 기반으로 한 모노레포 구조입니다.
- 백엔드가 실행되기 전까지는 프론트엔드에서 **Mock Fallback**이 동작하여 전체 흐름을 시연해볼 수 있습니다.
- 모든 업로드 이미지는 `apps/api/data/uploads`에 저장됩니다. (git-ignored)
